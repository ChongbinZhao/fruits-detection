### 简介

- 此项目是对特定种类的水果进行目标检测，涉及到很多传统的数字图像处理方法，包括：
  - 归一化的色差法灰度化
  - `Otsu`算法
  - 孔洞填充算法
  - 开启闭合算法
  - 小波变换算法
  - 霍夫圆检测算法

- 这是一个针对特定目标的传统目标检测算法，检测精度不是特别高，同时它的泛化能力也不太好，这是后期我们可以优化的方向。
- 实现原理和检测效果请查看下文。
